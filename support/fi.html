Ominaisuudet

Tilit

  • Useiden pankkitilien manuaalinen seuranta
  • Henkilökohtaiset ja jaetut tilit
  • Saldo päivittyy reaaliajassa
  • Tapahtumahistoria
  • Muokattavat tilikuvakkeet
  • Manuaaliset saldon korjaukset

Lainat

  • Asunto-, auto-, opinto-, kulutus- ja henkilölainat, sekä luottokortit
  • Yksityiskohtaiset maksuaikataulut
  • Korkolaskenta
  • Kuukausittaisten maksujen seuranta
  • Annuitetti- ja tasalyhennyslainojen tuki
  • Automaattinen maksujen käsittely
  • Tarkat maksukirjaukset
  • Toistuvan luottokorttikäytön seuranta

Tapahtumat

  • Tulojen ja menojen kirjaaminen
  • Tapahtumien kategorisointi
  • Toistuvat tapahtumat
  • Yksityiskohtainen historia
  • Siirrot tilien välillä
  • Vapaavalintaiset päivämäärät
  • Haku ja suodatus

Datanhallinta

  • Vienti .regnskappdata-varmuuskopiotiedostoiksi
  • Valikoiva vienti (valitse sisällytettävä data)
  • Tuonti varmuuskopiosta
  • Korvaa tai yhdistä tuonnin yhteydessä
  • Paikallinen tallennus – ei ulkoisia palvelimia

Lisäominaisuuksia

  • Useita valuuttoja (NOK, USD, EUR, GBP, SEK, DKK)
  • Monikielisyystuki
  • Muokattavat kategoriat ja kuvakkeet
  • Taloudellinen yleiskuva ja tilastot
  • Muokattava palkanmaksupäivä
  • Biometrinen tunnistus (Face ID/Touch ID)
  • “Make it Rain” -animaatio
  • Senttien näyttö/piilotus

Usein kysyttyä

Miten lisään uuden tilin?

  1. Avaa Tilit-välilehti
  2. Napauta + -painiketta
  3. Syötä tilin tiedot
  4. Tallenna

Miten perustan lainan?

  1. Avaa Lainat-välilehti
  2. Napauta +
  3. Valitse lainatyyppi
  4. Syötä laina­tiedot
  5. Tallenna

Miten teen toistuvan tapahtuman?

  1. Lisää uusi tapahtuma
  2. Ota “Toistuva” käyttöön
  3. Valitse toistuvuus
  4. Valitse aloitus- ja mahdollinen päättymispäivä
  5. Tallenna

Miten vien datani varmuuskopioksi?

  1. Avaa Asetukset
  2. Valitse “Vienti ja tuonti”
  3. Valitse “Vienti”-välilehti
  4. Valitse vietävä data
  5. Napauta “Vie data”
  6. Valitse tallennuspaikka

Miten tuon varmuuskopion?

  1. Avaa Asetukset
  2. Valitse “Vienti ja tuonti”
  3. Valitse “Tuonti”-välilehti
  4. Valitse korvaus tai yhdistäminen
  5. Valitse varmuuskopiotiedosto
  6. Vahvista tuonti

Miten vaihdan kielen?

  1. Avaa Asetukset
  2. Valitse Kieli
  3. Valitse haluamasi kieli

Miten otan Face ID/Touch ID:n käyttöön?

  1. Paina pitkään sovelluskuvaketta
  2. Valitse “Require Face ID/Touch ID”

Miten vaihdan valuutan?

  1. Avaa Asetukset
  2. Valitse Valuutta
  3. Valitse haluamasi valuutta

Miten asetan palkanmaksupäivän?

  1. Avaa Asetukset
  2. Etsi Palkanmaksupäivä
  3. Valitse vakiopäivä (1–31); viikonloput huomioidaan automaattisesti

Dataturva

  • Kaikki data tallennetaan paikallisesti laitteellesi (SwiftData)
  • Ei tarvitse internet-yhteyttä
  • Ei käyttäjätiliä
  • Face ID/Touch ID -tuki
  • Manuaalinen vienti/tuonti antaa täyden hallinnan
  • iOS salaa taloustietosi laitetasolla
