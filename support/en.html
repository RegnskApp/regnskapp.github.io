Features

Accounts

  • Track multiple bank accounts manually
  • Support for shared and personal accounts
  • Real-time balance updates
  • Transaction history
  • Customizable account icons
  • Manual balance adjustments

Loans

  • Support for various loan types: mortgages, car loans, student loans, personal loans, consumer loans, credit cards
  • Detailed payment schedules
  • Interest calculations
  • Monthly payment tracking
  • Support for both annuity and serial loan types
  • Automatic payment processing
  • Detailed loan payment records
  • Recurring credit card usage tracking

Transactions

  • Record income and expenses
  • Categorize transactions
  • Support for recurring transactions
  • Detailed transaction history
  • Transfer between accounts
  • Custom transaction dates
  • Transaction search and filtering

Data Management

  • Export data to .regnskappdata files for backup
  • Selective data export (choose what to include)
  • Import data from backup files
  • Option to replace or merge data during import
  • Local data storage with no external servers

Additional Features

  • Multiple currency support (NOK, USD, EUR, GBP, SEK, DKK)
  • Multi-language support (Norwegian, English, French, Spanish, German, Danish, Swedish)
  • Custom categories with icons

  • Biometric authentication (Face ID/Touch ID)

Frequently Asked Questions

How do I add a new account?

  1. Go to the Accounts tab
  2. Tap the + button
  3. Enter account details
  4. Save

How do I set up a loan?

  1. Go to the Loans tab
  2. Tap the + button
  3. Choose loan type
  4. Enter loan details
  5. Save

How do I set up a recurring transaction?

  1. Add a new transaction
  2. Enable the "Recurring" toggle
  3. Set the frequency (weekly, monthly, yearly)
  4. Choose the start and optional end date
  5. Save the transaction

How do I export my data for backup?

  1. Go to Settings
  2. Select "Export & Import"
  3. Choose the "Export" tab
  4. Select what data you want to include
  5. Tap "Export Data"
  6. Choose where to save the .regnskappdata file

How do I import data from a backup?

  1. Go to Settings
  2. Select "Export & Import"
  3. Choose the "Import" tab
  4. Decide whether to replace existing data
  5. Tap to select your backup file
  6. Confirm the import

How do I change the language?

  1. Go to Settings
  2. Select Language
  3. Choose your preferred language (Norwegian, English, French, Spanish, German, Danish, or Swedish)
  4. Restart the app if prompted

How do I enable Face ID/Touch ID?

  1. Long press the app icon on your home screen
  2. Select "Require Face ID/Touch ID"

How do I change the currency?

  1. Go to Settings
  2. Tap on Currency
  3. Select your preferred currency from the list

How do I set my payday date?

  1. Go to Settings
  2. Find the Payday setting
  3. Select your regular payday date (1-31)
  4. The system will automatically adjust for weekends

Data Security

  • All data is stored locally on your device using SwiftData
  • No internet connection required
  • No account creation needed
  • Support for Face ID/Touch ID authentication
  • Manual data export and import gives you complete control
  • Your financial data is encrypted at the device level by iOS
