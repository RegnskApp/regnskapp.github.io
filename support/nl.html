Veelgestelde vragen

Hoe voeg ik een nieuwe rekening toe?

  1. Ga naar de tab Rekeningen
  2. Tik op +
  3. Vul de gegevens in
  4. Sla op

Hoe stel ik een lening in?

  1. Ga naar de tab Leningen
  2. Tik op +
  3. Kies het lenings­type
  4. Vul de details in
  5. Sla op

Hoe maak ik een terugkerende transactie?

  1. Voeg een nieuwe transactie toe
  2. Schakel “Terugkerend” in
  3. Kies de frequentie
  4. Kies start- en (optioneel) einddatum
  5. Sla op

Hoe exporteer ik mijn data voor back‑up?

  1. Ga naar Instellingen
  2. Kies “Export & Import”
  3. Kies de tab “Export”
  4. Maak je selectie
  5. Tik op “Export Data”
  6. Kies waar je het .regnskappdata‑bestand opslaat

Hoe importeer ik data uit een back‑up?

  1. Ga naar Instellingen
  2. Kies “Export & Import”
  3. Kies de tab “Import”
  4. Kies vervangen of samenvoegen
  5. Selecteer je back‑upbestand
  6. Bevestig de import

Hoe verander ik de taal?

  1. Ga naar Instellingen
  2. Kies Taal
  3. Kies je voorkeurstaal

Hoe schakel ik Face ID/Touch ID in?

  1. Druk lang op het app‑icoon op je beginscherm
  2. Kies “Require Face ID/Touch ID”

Hoe verander ik de valuta?

  1. Ga naar Instellingen
  2. Tik op Valuta
  3. Kies je valuta uit de lijst

Hoe stel ik mijn betaaldatum in?

  1. Ga naar Instellingen
  2. Zoek de instelling Betaaldag
  3. Kies een vaste dag (1–31); weekends worden automatisch verwerkt

Gegevensbeveiliging
