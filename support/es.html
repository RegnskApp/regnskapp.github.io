Funciones

Cuentas

  • Realice seguimiento manual de múltiples cuentas bancarias
  • Soporte para cuentas compartidas y personales
  • Actualizaciones de saldo en tiempo real
  • Historial de transacciones
  • Iconos de cuenta personalizables
  • Ajustes manuales de saldo

Préstamos

  • Soporte para varios tipos de préstamos: hipotecas, préstamos para automóvil, préstamos estudiantiles, préstamos personales, préstamos de consumo, tarjetas de crédito
  • Calendarios de pago detallados
  • Cálculo de intereses
  • Seguimiento de pagos mensuales
  • Soporte para préstamos de tipo anualidad y préstamos de tipo serial
  • Procesamiento automático de pagos
  • Registros detallados de pagos de préstamos
  • Seguimiento del uso recurrente de tarjetas de crédito

Transacciones

  • Registrar ingresos y gastos
  • Categorizar transacciones
  • Soporte para transacciones recurrentes
  • Historial detallado de transacciones
  • Transferencias entre cuentas
  • Fechas de transacción personalizadas
  • Búsqueda y filtrado de transacciones

Gestión de datos

  • Exportar datos a archivos .regnskappdata para copias de seguridad
  • Exportación selectiva de datos
  • Importar datos desde archivos de respaldo
  • Opción para reemplazar o combinar datos durante la importación
  • Almacenamiento local de datos sin servidores externos

Funciones adicionales

  • Soporte para múltiples monedas (más de 160 monedas)
  • Soporte multilenguaje (árabe, chino simplificado, chino tradicional, danés, neerlandés, inglés, finlandés, francés, alemán, hindi, italiano, japonés, noruego, español, sueco, ucraniano)
  • Categorías personalizadas con iconos
  • Resumen financiero y estadísticas
  • Configuración personalizable del día de pago
  • Autenticación biométrica (por ejemplo Face ID o Touch ID en iOS, huella digital o desbloqueo facial en Android)
  • Función de animación "Make it Rain" con opciones personalizables
  • Mostrar u ocultar céntimos en los importes

Preguntas frecuentes

¿Cómo agrego una nueva cuenta?

  1. Ir a la pestaña Cuentas
  2. Pulsar el botón +
  3. Introducir los detalles de la cuenta
  4. Guardar

¿Cómo configuro un préstamo?

  1. Ir a la pestaña Préstamos
  2. Pulsar el botón +
  3. Elegir el tipo de préstamo
  4. Introducir los detalles del préstamo
  5. Guardar

¿Cómo configuro una transacción recurrente?

  1. Agregar una nueva transacción
  2. Activar el interruptor "Recurring"
  3. Establecer la frecuencia (semanal, mensual, anual)
  4. Elegir la fecha de inicio y opcionalmente la fecha de finalización
  5. Guardar la transacción

¿Cómo exporto mis datos para una copia de seguridad?

  1. Ir a Configuración
  2. Seleccionar "Export & Import"
  3. Elegir la pestaña "Export"
  4. Seleccionar qué datos incluir
  5. Pulsar "Export Data"
  6. Elegir dónde guardar el archivo .regnskappdata

¿Cómo importo datos desde una copia de seguridad?

  1. Ir a Configuración
  2. Seleccionar "Export & Import"
  3. Elegir la pestaña "Import"
  4. Decidir si se deben reemplazar los datos existentes
  5. Pulsar para seleccionar el archivo de respaldo
  6. Confirmar la importación

¿Cómo cambio el idioma?

  1. Ir a Configuración
  2. Seleccionar Idioma
  3. Elegir el idioma preferido
  4. Reiniciar la aplicación si se solicita

¿Cómo habilito el bloqueo biométrico?

iOS: Mantenga presionado el icono de la aplicación en la pantalla de inicio y seleccione "Require Face ID" o "Require Touch ID".

Android: Abra la aplicación, vaya a Configuración, luego Seguridad o Bloqueo de aplicación y active el bloqueo biométrico.

¿Cómo cambio la moneda?

  1. Ir a Configuración
  2. Pulsar en Moneda
  3. Seleccionar la moneda preferida de la lista

¿Cómo configuro mi fecha de pago?

  1. Ir a Configuración
  2. Buscar la configuración Día de pago
  3. Ajustar las opciones
  4. El sistema puede ajustarse automáticamente para fines de semana

Seguridad de datos

  • Todos los datos se almacenan localmente en su dispositivo. En iOS la aplicación utiliza SwiftData; en Android utiliza almacenamiento de base de datos local
  • No se requiere conexión a internet
  • No se requiere creación de cuenta
  • Soporte para autenticación biométrica
  • La exportación e importación manual de datos le da control total
  • Sus datos financieros están protegidos por el cifrado integrado del dispositivo

Actualizaciones

La aplicación se actualiza regularmente con mejoras y nuevas funciones. Asegúrese de mantener la aplicación actualizada a través de App Store o Google Play.
