  • Опція замінити або об'єднати дані під час імпорту

  • Функція анімації "Зроби дощ" з налаштовуваними опціями
  • Показати/приховати копійки в сумах

Поширені запитання

Як додати новий обліковий запис?

  1. Перейдіть до вкладки Облікові записи
  2. Торкніться кнопки +
  3. Введіть дані облікового запису
  4. Зберегти

Як налаштувати позику?

  1. Перейдіть до вкладки Позики
  2. Торкніться кнопки +
  3. Виберіть тип позики
  4. Введіть деталі позики
  5. Зберегти

Як налаштувати повторювану транзакцію?

  1. Додайте нову транзакцію
  2. Увімкніть перемикач "Повторювана"
  3. Встановіть частоту (щотижнева, щомісячна, щорічна)
  4. Виберіть дату початку та необов'язкову дату завершення
  5. Збережіть транзакцію

Як експортувати мої дані для резервного копіювання?

  1. Перейдіть до Налаштувань
  2. Виберіть "Експорт та імпорт"
  3. Виберіть вкладку "Експорт"
  4. Виберіть, які дані ви хочете включити
  5. Торкніться "Експортувати дані"
  6. Виберіть, де зберегти файл .regnskappdata

Як імпортувати дані з резервної копії?

  1. Перейдіть до Налаштувань
  2. Виберіть "Експорт та імпорт"
  3. Виберіть вкладку "Імпорт"
  4. Вирішіть, чи замінити існуючі дані
  5. Торкніться, щоб вибрати файл резервної копії
  6. Підтвердіть імпорт

Як змінити мову?

  1. Перейдіть до Налаштувань
  2. Виберіть Мову
  3. Виберіть бажану мову (норвезька, англійська, французька, іспанська, німецька, данська або шведська)
  4. Перезапустіть додаток, якщо буде запропоновано

Як увімкнути Face ID/Touch ID?

  1. Довго натисніть іконку додатка на головному екрані
  2. Виберіть "Вимагати Face ID/Touch ID"

Як змінити валюту?

  1. Перейдіть до Налаштувань
  2. Торкніться Валюти
  3. Виберіть бажану валюту зі списку

Як встановити дату виплати?

  1. Перейдіть до Налаштувань
  2. Знайдіть налаштування Дня виплати
  3. Виберіть регулярну дату виплати (1-31)
  4. Система автоматично скоригує для вихідних

Безпека даних

  • Усі дані зберігаються локально на вашому пристрої за допомогою SwiftData
  • Не потрібне інтернет-з'єднання
  • Не потрібно створювати обліковий запис
  • Підтримка аутентифікації Face ID/Touch ID
  • Ручний експорт та імпорт даних дає вам повний контроль
  • Ваші фінансові дані шифруються на рівні пристрою iOS
