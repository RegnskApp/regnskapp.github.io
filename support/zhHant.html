功能

帳戶

  • 手動追蹤多個銀行帳戶
  • 支援個人與共享帳戶
  • 即時更新餘額
  • 交易記錄
  • 可自訂帳戶圖示
  • 手動調整餘額

貸款

  • 房貸、車貸、學貸、個人貸、消費貸、信用卡
  • 詳細的還款時程
  • 利息計算
  • 每月付款追蹤
  • 支援本息均攤與本金均攤
  • 自動處理付款
  • 詳細的貸款付款紀錄
  • 追蹤信用卡的週期性使用

交易

  • 記錄收入與支出
  • 交易分類
  • 支援週期性交易
  • 詳細的交易歷史
  • 帳戶間轉帳
  • 自訂日期
  • 搜尋與篩選

資料管理

  • 匯出為 .regnskappdata 備份檔
  • 選擇性匯出（自選內容）
  • 自備份匯入資料
  • 匯入時可選擇取代或合併
  • 本機儲存，無外部伺服器

其他

  • 多種貨幣（NOK、USD、EUR、GBP、SEK、DKK）
  • 多語系支援
  • 可自訂分類與圖示
  • 財務總覽與統計
  • 可自訂發薪日
  • 生物辨識（Face ID/Touch ID）
  • “Make it Rain” 動畫
  • 顯示/隱藏角分

常見問題

如何新增帳戶？

  1. 前往「帳戶」分頁
  2. 點擊 +
  3. 輸入帳戶資料
  4. 儲存

如何設定貸款？

  1. 前往「貸款」分頁
  2. 點擊 +
  3. 選擇貸款類型
  4. 輸入詳細資料
  5. 儲存

如何建立週期性交易？

  1. 新增交易
  2. 啟用「週期性」
  3. 設定頻率（週/月/年）
  4. 選擇開始與可選的結束日期
  5. 儲存

如何匯出備份？

  1. 開啟「設定」
  2. 選擇「匯出與匯入」
  3. 前往「匯出」分頁
  4. 選擇要包含的資料
  5. 點擊「匯出資料」
  6. 儲存 .regnskappdata 檔案

如何自備份匯入？

  1. 開啟「設定」
  2. 選擇「匯出與匯入」
  3. 前往「匯入」分頁
  4. 選擇取代或合併
  5. 選擇備份檔
  6. 確認匯入

如何變更語言？

  1. 開啟「設定」
  2. 選擇「語言」
  3. 選擇偏好的語言

如何啟用 Face ID/Touch ID？

  1. 在主畫面長按 App 圖示
  2. 選擇「Require Face ID/Touch ID」

如何變更貨幣？

  1. 開啟「設定」
  2. 點擊「貨幣」
  3. 從清單中選擇

如何設定發薪日？

  1. 開啟「設定」
  2. 找到「發薪日」設定
  3. 選擇日期（1–31），週末自動調整

資料安全

  • 所有資料以 SwiftData 儲存在本機
  • 不需要網路連線
  • 不需要建立帳號
  • 支援 Face ID/Touch ID
  • 手動匯出/匯入，完整掌控
  • iOS 在裝置層級加密財務資料
