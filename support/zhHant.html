功能

帳戶

  • 手動追蹤多個銀行帳戶
  • 支援共享帳戶與個人帳戶
  • 即時餘額更新
  • 交易紀錄
  • 可自訂帳戶圖示
  • 手動調整餘額

貸款

  • 支援多種貸款類型：房貸、汽車貸款、學生貸款、個人貸款、消費貸款、信用卡
  • 詳細的還款計畫
  • 利息計算
  • 每月付款追蹤
  • 支援年金貸款與等額本金貸款
  • 自動付款處理
  • 詳細的貸款付款紀錄
  • 追蹤信用卡的定期使用情況

交易

  • 記錄收入與支出
  • 為交易分類
  • 支援重複交易
  • 詳細的交易紀錄
  • 帳戶之間轉帳
  • 自訂交易日期
  • 交易搜尋與篩選

資料管理

  • 將資料匯出為 .regnskappdata 檔案以進行備份
  • 選擇性資料匯出（選擇要包含的內容）
  • 從備份檔案匯入資料
  • 匯入時可選擇取代或合併資料
  • 本地資料儲存，不使用外部伺服器

其他功能

  • 支援多種貨幣（超過160種貨幣）
  • 多語言支援（阿拉伯語、簡體中文、繁體中文、丹麥語、荷蘭語、英語、芬蘭語、法語、德語、印地語、義大利語、日語、挪威語、西班牙語、瑞典語、烏克蘭語）
  • 具有圖示的自訂分類
  • 財務總覽與統計
  • 可自訂發薪日設定
  • 生物識別驗證（例如 iOS 的 Face ID 或 Touch ID，Android 的指紋或臉部解鎖）
  • 具有可自訂選項的「Make it Rain」動畫功能
  • 顯示或隱藏金額中的分

常見問題

如何新增一個帳戶？

  1. 前往 Accounts 分頁
  2. 點擊 + 按鈕
  3. 輸入帳戶資訊
  4. 儲存

如何設定貸款？

  1. 前往 Loans 分頁
  2. 點擊 + 按鈕
  3. 選擇貸款類型
  4. 輸入貸款詳細資料
  5. 儲存

如何設定重複交易？

  1. 新增一筆交易
  2. 啟用「Recurring」開關
  3. 設定頻率（每週、每月、每年）
  4. 選擇開始日期與可選的結束日期
  5. 儲存交易

如何匯出資料進行備份？

  1. 前往 Settings
  2. 選擇「Export, Import & PDF」
  3. 選擇「Export」分頁
  4. 選擇要包含的資料
  5. 點擊「Export Data」
  6. 選擇 .regnskappdata 檔案的儲存位置

如何從備份匯入資料？

  1. 前往 Settings
  2. 選擇「Export, Import & PDF」
  3. 選擇「Import」分頁
  4. 選擇備份檔案
  5. 確認匯入
  6. 在匯入之前，所有現有資料將被刪除

如何更改語言？

  1. 前往 Settings
  2. 選擇 Language
  3. 選擇您偏好的語言
  4. 如果提示，重新啟動應用程式

如何啟用生物識別鎖？

iOS：在主畫面長按應用程式圖示，然後選擇「Require Face ID」或「Require Touch ID」。

Android：開啟應用程式，前往 Settings，然後進入 Security 或 App lock 並啟用生物識別鎖。

如何更改貨幣？

  1. 前往 Settings
  2. 點擊 Currency
  3. 從清單中選擇您偏好的貨幣

如何設定發薪日？

  1. 前往 Settings
  2. 找到 Payday 設定
  3. 調整選項以設定您的固定發薪日
  4. 系統可以自動根據週末進行調整

資料安全

  • 所有資料都儲存在您的裝置本地。在 iOS 上應用程式使用 SwiftData；在 Android 上使用本地資料庫儲存
  • 不需要網際網路連線
  • 不需要建立帳戶
  • 支援生物識別驗證
  • 手動匯出與匯入資料讓您擁有完全控制權
  • 您的財務資料受到裝置內建加密的保護

更新

應用程式會定期更新以提供改進和新功能。請確保透過 App Store（iOS）或 Google Play（Android）保持應用程式為最新版本。
