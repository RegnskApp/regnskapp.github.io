功能

帳戶

  • 手動追蹤多個銀行帳戶
  • 支援共享和個人帳戶
  • 即時餘額更新
  • 交易歷史
  • 可自訂帳戶圖示
  • 手動餘額調整

貸款

  • 支援各種貸款類型：抵押貸款、汽車貸款、學生貸款、個人貸款、消費貸款、信用卡
  • 詳細的還款計劃
  • 利息計算
  • 每月還款追蹤
  • 支援年金和系列貸款類型
  • 自動還款處理
  • 詳細的貸款還款記錄
  • 循環信用卡使用追蹤

交易

  • 記錄收入和支出
  • 分類交易
  • 支援定期交易
  • 詳細的交易歷史
  • 帳戶間轉帳
  • 自訂交易日期
  • 交易搜尋和篩選

資料管理

  • 匯出資料到.regnskappdata檔案進行備份
  • 選擇性資料匯出（選擇要包含的內容）
  • 從備份檔案匯入資料
  • 匯入時選擇替換或合併資料的選項
  • 本機資料儲存，無需外部伺服器

附加功能

  • 多種貨幣支援（NOK、USD、EUR、GBP、SEK、DKK）
  • 多語言支援（挪威語、英語、法語、西班牙語、德語、丹麥語、瑞典語）
  • 帶圖示的自訂類別
  • 財務概覽和統計
  • 可自訂的發薪日設定
  • 生物識別認證（Face ID/Touch ID）
  • "下雨"動畫功能，可自訂選項
  • 顯示/隱藏金額中的分幣

常見問題

如何新增帳戶？

  1. 前往帳戶分頁
  2. 點擊+按鈕
  3. 輸入帳戶詳細資訊
  4. 儲存

如何設定貸款？

  1. 前往貸款分頁
  2. 點擊+按鈕
  3. 選擇貸款類型
  4. 輸入貸款詳細資訊
  5. 儲存

如何設定定期交易？

  1. 新增交易
  2. 啟用"定期"切換開關
  3. 設定頻率（每週、每月、每年）
  4. 選擇開始日期和可選的結束日期
  5. 儲存交易

如何匯出資料進行備份？

  1. 前往設定
  2. 選擇"匯出和匯入"
  3. 選擇"匯出"分頁
  4. 選擇要包含的資料
  5. 點擊"匯出資料"
  6. 選擇儲存.regnskappdata檔案的位置

如何從備份匯入資料？

  1. 前往設定
  2. 選擇"匯出和匯入"
  3. 選擇"匯入"分頁
  4. 決定是否替換現有資料
  5. 點擊選擇備份檔案
  6. 確認匯入

如何變更語言？

  1. 前往設定
  2. 選擇語言
  3. 選擇您的首選語言（挪威語、英語、法語、西班牙語、德語、丹麥語或瑞典語）
  4. 如果提示，請重新啟動應用程式

如何啟用Face ID/Touch ID？

  1. 長按主畫面上的應用程式圖示
  2. 選擇"需要Face ID/Touch ID"

如何變更貨幣？

  1. 前往設定
  2. 點擊貨幣
  3. 從清單中選擇您的首選貨幣

如何設定發薪日？

  1. 前往設定
  2. 找到發薪日設定
  3. 選擇您的定期發薪日（1-31）
  4. 系統將自動調整週末

資料安全

  • 所有資料都使用SwiftData在本機儲存在您的裝置上
  • 無需網際網路連線
  • 無需建立帳戶
  • 支援Face ID/Touch ID認證
  • 手動資料匯出和匯入讓您完全控制
  • 您的財務資料在iOS裝置層級加密
