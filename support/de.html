Häufig gestellte Fragen

Wie füge ich ein neues Konto hinzu?

  1. Gehen Sie zum Konten-Tab
  2. Tippen Sie auf die +-Schaltfläche
  3. Geben Sie Kontodetails ein
  4. Speichern

Wie richte ich einen Kredit ein?

  1. Gehen Sie zum Kredite-Tab
  2. Tippen Sie auf die +-Schaltfläche
  3. Wählen Sie den Kredittyp
  4. Geben Sie Kreditdetails ein
  5. Speichern

Wie richte ich eine wiederkehrende Transaktion ein?

  1. Fügen Sie eine neue Transaktion hinzu
  2. Aktivieren Sie den "Wiederkehrend"-Schalter
  3. Stellen Sie die Häufigkeit ein (wöchentlich, monatlich, jährlich)
  4. Wählen Sie das Start- und optionale Enddatum
  5. Speichern Sie die Transaktion

Wie exportiere ich meine Daten zur Sicherung?

  1. Gehen Sie zu den Einstellungen
  2. Wählen Sie "Exportieren und Importieren"
  3. Wählen Sie den "Exportieren"-Tab
  4. Wählen Sie aus, welche Daten eingeschlossen werden sollen
  5. Tippen Sie auf "Daten exportieren"
  6. Wählen Sie, wo die .regnskappdata-Datei gespeichert werden soll

Wie importiere ich Daten aus einer Sicherung?

  1. Gehen Sie zu den Einstellungen
  2. Wählen Sie "Exportieren und Importieren"
  3. Wählen Sie den "Importieren"-Tab
  4. Entscheiden Sie, ob vorhandene Daten ersetzt werden sollen
  5. Tippen Sie, um Ihre Sicherungsdatei auszuwählen
  6. Bestätigen Sie den Import

Wie ändere ich die Sprache?

  1. Gehen Sie zu den Einstellungen
  2. Wählen Sie Sprache
  3. Wählen Sie Ihre bevorzugte Sprache (Norwegisch, Englisch, Französisch, Spanisch, Deutsch, Dänisch oder Schwedisch)
  4. Starten Sie die App neu, falls aufgefordert

Wie aktiviere ich Face ID/Touch ID?

  1. Drücken Sie lange auf das App-Symbol auf Ihrem Startbildschirm
  2. Wählen Sie "Face ID/Touch ID erforderlich"

Wie ändere ich die Währung?

  1. Gehen Sie zu den Einstellungen
  2. Tippen Sie auf Währung
  3. Wählen Sie Ihre bevorzugte Währung aus der Liste

Wie stelle ich mein Gehaltstag-Datum ein?

  1. Gehen Sie zu den Einstellungen
  2. Suchen Sie die Gehaltstag-Einstellung
  3. Wählen Sie Ihr regelmäßiges Gehaltstag-Datum (1-31)
  4. Das System wird automatisch für Wochenenden angepasst

Datensicherheit

  • Alle Daten werden lokal auf Ihrem Gerät mit SwiftData gespeichert
  • Keine Internetverbindung erforderlich
  • Keine Kontoerstellung notwendig
  • Unterstützung für Face ID/Touch ID-Authentifizierung
  • Manueller Datenexport und -import gibt Ihnen vollständige Kontrolle
  • Ihre Finanzdaten werden auf Geräteebene von iOS verschlüsselt
