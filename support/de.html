Funktionen

Konten

  • Mehrere Bankkonten manuell verfolgen
  • Unterstützung für gemeinsame und persönliche Konten
  • Echtzeit Aktualisierung des Kontostands
  • Transaktionsverlauf
  • Anpassbare Kontosymbole
  • Manuelle Anpassung des Kontostands

Kredite

  • Unterstützung für verschiedene Kreditarten: Hypotheken, Autokredite, Studienkredite, Privatkredite, Konsumentenkredite, Kreditkarten
  • Detaillierte Zahlungspläne
  • Zinsberechnungen
  • Monatliche Zahlungsübersicht
  • Unterstützung für Annuitätenkredite und serielle Kredite
  • Automatische Zahlungsabwicklung
  • Detaillierte Kreditaufzeichnungen
  • Verfolgung wiederkehrender Kreditkartennutzung

Transaktionen

  • Einnahmen und Ausgaben erfassen
  • Transaktionen kategorisieren
  • Unterstützung für wiederkehrende Transaktionen
  • Detaillierter Transaktionsverlauf
  • Überweisungen zwischen Konten
  • Individuelle Transaktionsdaten
  • Transaktionssuche und Filterung

Datenverwaltung

  • Daten in .regnskappdata Dateien für Backups exportieren
  • Selektiver Datenexport
  • Daten aus Sicherungsdateien importieren
  • Option zum Ersetzen oder Zusammenführen von Daten beim Import
  • Lokale Datenspeicherung ohne externe Server

Weitere Funktionen

  • Unterstützung für mehrere Währungen (über 160 Währungen)
  • Unterstützung für mehrere Sprachen (Arabisch, Chinesisch vereinfacht, Chinesisch traditionell, Dänisch, Niederländisch, Englisch, Finnisch, Französisch, Deutsch, Hindi, Italienisch, Japanisch, Norwegisch, Spanisch, Schwedisch, Ukrainisch)
  • Benutzerdefinierte Kategorien mit Symbolen

  • Biometrische Authentifizierung (z. B. Face ID oder Touch ID auf iOS, Fingerabdruck oder Gesichtsentsperrung auf Android)
  • "Make it Rain" Animationsfunktion mit anpassbaren Optionen
  • Cents in Beträgen anzeigen oder ausblenden

Häufig gestellte Fragen

Wie füge ich ein neues Konto hinzu?

  1. Gehen Sie zum Konten Tab
  2. Tippen Sie auf die + Taste
  3. Geben Sie die Kontodaten ein
  4. Speichern

Wie richte ich einen Kredit ein?

  1. Gehen Sie zum Kredite Tab
  2. Tippen Sie auf die + Taste
  3. Wählen Sie den Kredittyp
  4. Geben Sie die Kreditdetails ein
  5. Speichern

Wie richte ich eine wiederkehrende Transaktion ein?

  1. Fügen Sie eine neue Transaktion hinzu
  2. Aktivieren Sie den "Recurring" Schalter
  3. Legen Sie die Häufigkeit fest (wöchentlich, monatlich, jährlich)
  4. Wählen Sie Startdatum und optional ein Enddatum
  5. Speichern Sie die Transaktion

Wie exportiere ich meine Daten für ein Backup?

  1. Gehen Sie zu Einstellungen
  2. Wählen Sie "Export, Import & PDF"
  3. Wählen Sie den Tab "Export"
  4. Wählen Sie aus, welche Daten enthalten sein sollen
  5. Tippen Sie auf "Export Data"
  6. Wählen Sie, wo die .regnskappdata Datei gespeichert werden soll

Wie importiere ich Daten aus einem Backup?

  1. Gehen Sie zu Einstellungen
  2. Wählen Sie "Export, Import & PDF"
  3. Wählen Sie den Tab "Import"
  4. Wählen Sie Ihre Backup Datei aus
  5. Bestätigen Sie den Import
  6. Alle vorhandenen Daten werden vor dem Import gelöscht

Wie ändere ich die Sprache?

  1. Gehen Sie zu Einstellungen
  2. Wählen Sie Sprache
  3. Wählen Sie Ihre bevorzugte Sprache
  4. Starten Sie die App neu, wenn Sie dazu aufgefordert werden

Wie aktiviere ich biometrische Sperre?

iOS: Halten Sie das App Symbol auf dem Startbildschirm gedrückt und wählen Sie "Require Face ID" oder "Require Touch ID".

Android: Öffnen Sie die App, gehen Sie zu Einstellungen und dann zu Sicherheit oder App Sperre und aktivieren Sie die biometrische Sperre.

Wie ändere ich die Währung?

  1. Gehen Sie zu Einstellungen
  2. Tippen Sie auf Währung
  3. Wählen Sie Ihre bevorzugte Währung aus der Liste

Wie lege ich meinen Gehaltstag fest?

  1. Gehen Sie zu Einstellungen
  2. Finden Sie die Gehaltstag Einstellung
  3. Passen Sie die Optionen an
  4. Das System kann Wochenenden automatisch berücksichtigen

Datensicherheit

  • Alle Daten werden lokal auf Ihrem Gerät gespeichert. Auf iOS verwendet die App SwiftData; auf Android eine lokale Datenbank
  • Keine Internetverbindung erforderlich
  • Keine Kontoerstellung erforderlich
  • Unterstützung für biometrische Authentifizierung
  • Manueller Datenexport und Import gibt Ihnen vollständige Kontrolle
  • Ihre Finanzdaten sind durch die integrierte Verschlüsselung Ihres Geräts geschützt
