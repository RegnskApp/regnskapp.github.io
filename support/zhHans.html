功能

账户

贷款

  • 支持各种贷款类型：抵押贷款、汽车贷款、学生贷款、个人贷款、消费贷款、信用卡
  • 详细的还款计划
  • 利息计算
  • 每月还款跟踪
  • 支持年金和系列贷款类型
  • 自动还款处理
  • 详细的贷款还款记录
  • 循环信用卡使用跟踪

交易

  • 记录收入和支出
  • 分类交易
  • 支持定期交易
  • 详细的交易历史
  • 账户间转账
  • 自定义交易日期
  • 交易搜索和筛选

数据管理

  • 导出数据到.regnskappdata文件进行备份
  • 选择性数据导出（选择要包含的内容）
  • 从备份文件导入数据
  • 导入时选择替换或合并数据的选项
  • 本地数据存储，无需外部服务器

常见问题

如何添加新账户？

  1. 转到账户选项卡
  2. 点击+按钮
  3. 输入账户详细信息
  4. 保存

如何设置贷款？

  1. 转到贷款选项卡
  2. 点击+按钮
  3. 选择贷款类型
  4. 输入贷款详细信息
  5. 保存

如何设置定期交易？

  1. 添加新交易
  2. 启用"定期"切换开关
  3. 设置频率（每周、每月、每年）
  4. 选择开始日期和可选的结束日期
  5. 保存交易

如何导出数据进行备份？

  1. 转到设置
  2. 选择"导出和导入"
  3. 选择"导出"选项卡
  4. 选择要包含的数据
  5. 点击"导出数据"
  6. 选择保存.regnskappdata文件的位置

如何从备份导入数据？

  1. 转到设置
  2. 选择"导出和导入"
  3. 选择"导入"选项卡
  4. 决定是否替换现有数据
  5. 点击选择备份文件
  6. 确认导入

如何更改语言？

  1. 转到设置
  2. 选择语言
  3. 选择您的首选语言（挪威语、英语、法语、西班牙语、德语、丹麦语或瑞典语）
  4. 如果提示，请重新启动应用

如何启用Face ID/Touch ID？

  1. 长按主屏幕上的应用图标
  2. 选择"需要Face ID/Touch ID"

如何更改货币？

  1. 转到设置
  2. 点击货币
  3. 从列表中选择您的首选货币

如何设置发薪日？

  1. 转到设置
  2. 找到发薪日设置
  3. 选择您的常规发薪日（1-31）
  4. 系统将自动调整周末

数据安全

  • 所有数据都使用SwiftData本地存储在您的设备上
  • 无需互联网连接
  • 无需创建账户
  • 支持Face ID/Touch ID身份验证
  • 手动数据导出和导入让您完全控制
  • 您的财务数据在iOS设备层面加密
