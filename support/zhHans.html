功能

账户

  • 手动跟踪多个银行账户
  • 支持共享账户和个人账户
  • 实时余额更新
  • 交易历史
  • 可自定义账户图标
  • 手动余额调整

贷款

  • 支持多种贷款类型：房贷、汽车贷款、学生贷款、个人贷款、消费贷款、信用卡
  • 详细的还款计划
  • 利息计算
  • 每月还款跟踪
  • 支持等额本息和等额本金贷款类型
  • 自动付款处理
  • 详细的贷款还款记录
  • 跟踪信用卡的定期使用情况

交易

  • 记录收入和支出
  • 对交易进行分类
  • 支持定期交易
  • 详细的交易历史
  • 账户之间的转账
  • 自定义交易日期
  • 交易搜索和筛选

数据管理

  • 将数据导出为 .regnskappdata 文件以进行备份
  • 选择性数据导出（选择要包含的内容）
  • 从备份文件导入数据
  • 导入时可以选择替换或合并数据
  • 本地数据存储，无需外部服务器

其他功能

  • 支持多种货币（超过160种货币）
  • 多语言支持（阿拉伯语、简体中文、繁体中文、丹麦语、荷兰语、英语、芬兰语、法语、德语、印地语、意大利语、日语、挪威语、西班牙语、瑞典语、乌克兰语）
  • 带图标的自定义分类
  • 财务概览和统计
  • 可自定义发薪日设置
  • 生物识别身份验证（例如 iOS 上的 Face ID 或 Touch ID，Android 上的指纹或面部解锁）
  • 带有自定义选项的 “Make it Rain” 动画功能
  • 显示或隐藏金额中的分

常见问题

如何添加新账户？

  1. 进入 Accounts 选项卡
  2. 点击 + 按钮
  3. 输入账户信息
  4. 保存

如何设置贷款？

  1. 进入 Loans 选项卡
  2. 点击 + 按钮
  3. 选择贷款类型
  4. 输入贷款详情
  5. 保存

如何设置定期交易？

  1. 添加一笔新交易
  2. 启用 “Recurring” 开关
  3. 设置频率（每周、每月、每年）
  4. 选择开始日期和可选结束日期
  5. 保存交易

如何导出数据进行备份？

  1. 进入 Settings
  2. 选择 “Export, Import & PDF”
  3. 选择 “Export” 选项卡
  4. 选择要包含的数据
  5. 点击 “Export Data”
  6. 选择保存 .regnskappdata 文件的位置

如何从备份导入数据？

  1. 进入 Settings
  2. 选择 “Export, Import & PDF”
  3. 选择 “Import” 选项卡
  4. 选择备份文件
  5. 确认导入
  6. 在导入之前，所有现有数据将被删除

如何更改语言？

  1. 进入 Settings
  2. 选择 Language
  3. 选择您偏好的语言
  4. 如果提示，请重新启动应用

如何启用生物识别锁？

iOS：在主屏幕上长按应用图标，然后选择 “Require Face ID” 或 “Require Touch ID”。

Android：打开应用，进入 Settings，然后进入 Security 或 App lock，并启用生物识别锁。

如何更改货币？

  1. 进入 Settings
  2. 点击 Currency
  3. 从列表中选择您偏好的货币

如何设置发薪日？

  1. 进入 Settings
  2. 找到 Payday 设置
  3. 调整选项以设置您的常规发薪日
  4. 系统可以自动根据周末进行调整

数据安全

  • 所有数据都存储在您的设备本地。在 iOS 上应用使用 SwiftData；在 Android 上使用本地数据库存储
  • 无需互联网连接
  • 无需创建账户
  • 支持生物识别身份验证
  • 手动导出和导入数据让您拥有完全控制权
  • 您的财务数据受到设备内置加密的保护

更新
