功能

账户

  • 手动跟踪多个银行账户
  • 支持个人与共享账户
  • 余额实时更新
  • 交易历史
  • 可自定义账户图标
  • 手动余额调整

贷款

  • 房贷、车贷、助学贷、个人贷、消费贷、信用卡
  • 详细还款计划
  • 利息计算
  • 每月还款跟踪
  • 支持等额本息与等额本金
  • 自动处理还款
  • 详细还款记录
  • 跟踪信用卡的周期性使用

交易

  • 记录收入与支出
  • 交易分类
  • 支持周期性交易
  • 详细交易历史
  • 账户间转账
  • 自定义日期
  • 搜索与筛选

数据管理

  • 导出为 .regnskappdata 备份文件
  • 选择性导出（自选包含内容）
  • 从备份导入数据
  • 导入时可选择替换或合并
  • 本地存储，无外部服务器

常见问题

如何添加新账户？

  1. 进入“账户”标签
  2. 点击 +
  3. 填写账户信息
  4. 保存

如何设置贷款？

  1. 进入“贷款”标签
  2. 点击 +
  3. 选择贷款类型
  4. 填写详细信息
  5. 保存

如何创建周期性交易？

  1. 新增交易
  2. 启用“周期性”
  3. 设置频率（周/月/年）
  4. 选择开始与可选结束日期
  5. 保存

如何导出备份？

  1. 打开“设置”
  2. 选择“导出与导入”
  3. 进入“导出”标签
  4. 选择要包含的数据
  5. 点击“导出数据”
  6. 保存 .regnskappdata 文件

如何从备份导入？

  1. 打开“设置”
  2. 选择“导出与导入”
  3. 进入“导入”标签
  4. 选择替换或合并
  5. 选择备份文件
  6. 确认导入

如何更改语言？

  1. 打开“设置”
  2. 选择“语言”
  3. 选择偏好的语言

如何启用 Face ID/Touch ID？

  1. 在主屏幕长按 App 图标
  2. 选择 “Require Face ID/Touch ID”

如何更改货币？

  1. 打开“设置”
  2. 点击“货币”
  3. 从列表中选择

如何设置发薪日？

  1. 打开“设置”
  2. 找到“发薪日”设置
  3. 选择日期（1–31），周末自动调整

数据安全

  • 所有数据使用 SwiftData 保存在本地
  • 无需联网
  • 无需创建账号
  • 支持 Face ID/Touch ID
  • 手动导入导出，完全掌控
  • iOS 在设备层面加密财务数据
