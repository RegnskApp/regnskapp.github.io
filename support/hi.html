  • विस्तृत ऋण भुगतान रिकॉर्ड
  • रीक्यूरिंग क्रेडिट कार्ड उपयोग ट्रैकिंग

  • बाहरी सर्वरों के बिना स्थानीय डेटा संग्रहण

अतिरिक्त सुविधाएँ

  • एकाधिक मुद्राओं का समर्थन (NOK, USD, EUR, GBP, SEK, DKK)
  • बहुभाषी समर्थन (नॉर्वेजियन, इंग्लिश, फ्रेंच, स्पेनिश, जर्मन, डेनिश, स्वीडिश)
  • आइकन के साथ कस्टम श्रेणियाँ
  • वित्तीय अवलोकन और आँकड़े
  • अनुकूलन योग्य वेतन दिवस सेटिंग
  • बायोमेट्रिक प्रमाणीकरण (Face ID/Touch ID)
  • अनुकूलन योग्य विकल्पों के साथ "मेक इट रेन" एनिमेशन फीचर
  • राशियों में पैसे दिखाएँ/छिपाएँ

अक्सर पूछे जाने वाले प्रश्न

मैं एक नया खाता कैसे जोड़ूँ?

  1. "खाते" टैब में जाएँ
  2. + बटन दबाएँ
  3. खाता विवरण दर्ज करें
  4. सहेजें

मैं एक ऋण कैसे सेट करूँ?

  1. "ऋण" टैब में जाएँ
  2. + बटन दबाएँ
  3. ऋण प्रकार चुनें
  4. ऋण विवरण दर्ज करें
  5. सहेजें

मैं एक रीक्यूरिंग लेन-देन कैसे सेट करूँ?

  1. एक नया लेन-देन जोड़ें
  2. "रीक्यूरिंग" स्विच को सक्रिय करें
  3. फ्रीक्वेंसी निर्धारित करें (साप्ताहिक, मासिक, वार्षिक)
  4. प्रारंभ तिथि और वैकल्पिक समाप्ति तिथि चुनें
  5. लेन-देन सहेजें

मैं बैकअप के लिए अपना डेटा कैसे एक्सपोर्ट करूँ?

  1. सेटिंग्स में जाएँ
  2. "एक्सपोर्ट और इंपोर्ट" चुनें
  3. "एक्सपोर्ट" टैब चुनें
  4. चुनें कि कौन सा डेटा शामिल करना है
  5. "डेटा एक्सपोर्ट करें" दबाएँ
  6. चुनें कि .regnskappdata फ़ाइल कहाँ सहेजनी है

मैं एक बैकअप से डेटा कैसे इंपोर्ट करूँ?

  1. सेटिंग्स में जाएँ
  2. "एक्सपोर्ट और इंपोर्ट" चुनें
  3. "इंपोर्ट" टैब चुनें
  4. निर्णय करें कि मौजूदा डेटा को बदलना है या नहीं
  5. अपनी बैकअप फ़ाइल चुनने के लिए दबाएँ
  6. इंपोर्ट की पुष्टि करें

मैं भाषा कैसे बदलूँ?

  1. सेटिंग्स में जाएँ
  2. "भाषा" चुनें
  3. अपनी पसंदीदा भाषा चुनें (नॉर्वेजियन, इंग्लिश, फ्रेंच, स्पेनिश, जर्मन, डेनिश या स्वीडिश)
  4. यदि कहा जाए तो ऐप को पुनरारंभ करें

मैं Face ID/Touch ID कैसे सक्रिय करूँ?

  1. अपने होम स्क्रीन पर ऐप आइकन को लंबे समय तक दबाएँ
  2. "Face ID/Touch ID आवश्यक करें" चुनें

मैं मुद्रा कैसे बदलूँ?

  1. सेटिंग्स में जाएँ
  2. "मुद्रा" पर दबाएँ
  3. सूची से अपनी पसंदीदा मुद्रा चुनें

मैं अपना वेतन दिवस कैसे सेट करूँ?

  1. सेटिंग्स में जाएँ
  2. "वेतन दिवस" सेटिंग खोजें
  3. अपना नियमित वेतन दिवस चुनें (1-31)
  4. सिस्टम स्वचालित रूप से सप्ताहांत के लिए समायोजित हो जाएगा

डेटा सुरक्षा

  • सभी डेटा SwiftData का उपयोग करके आपके डिवाइस पर स्थानीय रूप से संग्रहीत है
  • इंटरनेट कनेक्शन की आवश्यकता नहीं
  • खाता निर्माण आवश्यक नहीं
  • Face ID/Touch ID प्रमाणीकरण समर्थन
  • मैन्युअल डेटा एक्सपोर्ट और इंपोर्ट आपको पूर्ण नियंत्रण देता है
  • आपका वित्तीय डेटा iOS द्वारा डिवाइस स्तर पर एन्क्रिप्टेड है

अपडेट
