機能

アカウント

  • 複数口座の手動トラッキング
  • 個人用・共有用の両方に対応
  • 残高のリアルタイム更新
  • 取引履歴
  • 口座アイコンのカスタマイズ
  • 残高の手動調整

ローン

  • 住宅・自動車・学生・個人・消費者ローン、クレジットカード
  • 詳細な支払スケジュール
  • 利息計算
  • 月々の支払いトラッキング
  • 元利均等・元金均等の両方式に対応
  • 支払いの自動処理
  • 詳細な返済履歴
  • クレジットカードの定期的な利用の追跡

取引

  • 収入と支出の記録
  • カテゴリ分け
  • 定期取引の対応
  • 詳細な取引履歴
  • 口座間の振替
  • カスタム日付
  • 検索とフィルタ

データ管理

  • .regnskappdata 形式でのエクスポート（バックアップ）
  • 必要なデータのみ選択してエクスポート
  • バックアップからのインポート
  • インポート時に置換またはマージを選択可能
  • データはローカル保存（外部サーバーなし）

その他

  • 複数通貨（NOK, USD, EUR, GBP, SEK, DKK）
  • 多言語対応
  • アイコン付きのカスタムカテゴリ
  • 財務サマリーと統計
  • 給料日のカスタマイズ
  • 生体認証（Face ID/Touch ID）
  • “Make it Rain” アニメーション
  • 小数の表示/非表示

よくある質問

新しい口座を追加するには？

  1. 口座タブへ
  2. ＋をタップ
  3. 口座情報を入力
  4. 保存

ローンを設定するには？

  1. ローンタブへ
  2. ＋をタップ
  3. ローンタイプを選択
  4. 詳細を入力
  5. 保存

定期取引を設定するには？

  1. 新規取引を追加
  2. 「定期」を有効化
  3. 頻度を設定（週・月・年）
  4. 開始日と任意の終了日
  5. 保存

バックアップ用にデータをエクスポートするには？

  1. 設定へ
  2. 「エクスポートとインポート」を選択
  3. 「エクスポート」タブへ
  4. 含めるデータを選択
  5. 「データをエクスポート」をタップ
  6. .regnskappdata ファイルを保存

バックアップからインポートするには？

  1. 設定へ
  2. 「エクスポートとインポート」を選択
  3. 「インポート」タブへ
  4. 既存データの置換またはマージを選択
  5. バックアップファイルを選択
  6. インポートを確定

言語を変更するには？

  1. 設定へ
  2. 「言語」を選択
  3. 希望の言語を選ぶ

Face ID/Touch ID を有効にするには？

  1. ホーム画面のアプリアイコンを長押し
  2. 「Require Face ID/Touch ID」を選択

通貨を変更するには？

  1. 設定へ
  2. 「通貨」をタップ
  3. リストから選択

給料日を設定するには？

  1. 設定へ
  2. 給料日設定を開く
  3. 日付（1〜31）を選択（週末は自動調整）

データの安全性

  • すべてのデータは SwiftData により端末内に保存
  • インターネット接続不要
  • アカウント作成不要
  • Face ID/Touch ID に対応
  • 手動のエクスポート/インポートで完全な制御
  • iOS による端末レベルの暗号化
