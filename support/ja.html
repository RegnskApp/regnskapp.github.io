機能

アカウント

  • 複数の銀行口座を手動で追跡
  • 共有アカウントと個人アカウントのサポート
  • リアルタイムの残高更新
  • 取引履歴
  • カスタマイズ可能なアカウントアイコン
  • 手動の残高調整

ローン

  • さまざまな種類のローンのサポート：住宅ローン、自動車ローン、学生ローン、個人ローン、消費者ローン、クレジットカード
  • 詳細な支払いスケジュール
  • 利息計算
  • 月次支払いの追跡
  • 年賦と元利均等の両方のローンの種類のサポート
  • 自動支払い処理
  • 詳細なローン支払い記録
  • 定期的なクレジットカード利用の追跡

取引

  • 収入と支出を記録
  • 取引を分類
  • 定期的な取引のサポート
  • 詳細な取引履歴
  • アカウント間の振替
  • カスタム取引日付
  • 取引の検索とフィルタリング

データ管理

  • バックアップ用にデータを.regnskappdataファイルにエクスポート
  • 選択的なデータエクスポート（含めるものを選択）
  • バックアップファイルからデータをインポート
  • インポート時にデータを置換またはマージするオプション
  • 外部サーバーなしのローカルデータストレージ

追加機能

  • 複数通貨のサポート（NOK、USD、EUR、GBP、SEK、DKK）
  • 多言語サポート（ノルウェー語、英語、フランス語、スペイン語、ドイツ語、デンマーク語、スウェーデン語）
  • アイコン付きのカスタムカテゴリ
  • 財務概要と統計
  • カスタマイズ可能な給与日設定
  • 生体認証（Face ID/Touch ID）
  • カスタマイズ可能なオプション付きの「Make it Rain」アニメーション機能
  • 金額のセントの表示/非表示

よくある質問

新しいアカウントを追加するにはどうすればいいですか？

  1. アカウントタブに移動
  2. +ボタンをタップ
  3. アカウントの詳細を入力
  4. 保存

ローンを設定するにはどうすればいいですか？

  1. ローンタブに移動
  2. +ボタンをタップ
  3. ローンの種類を選択
  4. ローンの詳細を入力
  5. 保存

定期的な取引を設定するにはどうすればいいですか？

  1. 新しい取引を追加
  2. 「定期的な」トグルを有効化
  3. 頻度を設定（週次、月次、年次）
  4. 開始日とオプションの終了日を選択
  5. 取引を保存

バックアップ用にデータをエクスポートするにはどうすればいいですか？

  1. 設定に移動
  2. 「エクスポートとインポート」を選択
  3. 「エクスポート」タブを選択
  4. 含めるデータを選択
  5. 「データをエクスポート」をタップ
  6. .regnskappdataファイルを保存する場所を選択

バックアップからデータをインポートするにはどうすればいいですか？

  1. 設定に移動
  2. 「エクスポートとインポート」を選択
  3. 「インポート」タブを選択
  4. 既存のデータを置き換えるかどうかを決定
  5. バックアップファイルを選択するためにタップ
  6. インポートを確認

言語を変更するにはどうすればいいですか？

  1. 設定に移動
  2. 言語を選択
  3. 希望する言語を選択（ノルウェー語、英語、フランス語、スペイン語、ドイツ語、デンマーク語、またはスウェーデン語）
  4. プロンプトが表示されたらアプリを再起動

Face ID/Touch IDを有効にするにはどうすればいいですか？

  1. ホーム画面のアプリアイコンを長押し
  2. 「Face ID/Touch IDを要求」を選択

通貨を変更するにはどうすればいいですか？

  1. 設定に移動
  2. 通貨をタップ
  3. リストから希望する通貨を選択

給与日の日付を設定するにはどうすればいいですか？

  1. 設定に移動
  2. 給与日設定を探す
  3. 定期的な給与日の日付を選択（1-31）
  4. システムは自動的に週末を調整します

データセキュリティ

  • すべてのデータはSwiftDataを使用してデバイスにローカルで保存されます
  • インターネット接続は必要ありません
  • アカウント作成は必要ありません
  • Face ID/Touch ID認証のサポート
  • 手動のデータエクスポートとインポートで完全な制御が可能
  • 財務データはiOSによってデバイスレベルで暗号化されます
