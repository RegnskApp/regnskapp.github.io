機能

アカウント

  • 複数の銀行口座を手動で管理
  • 共有アカウントおよび個人アカウントに対応
  • リアルタイムの残高更新
  • 取引履歴
  • カスタマイズ可能なアカウントアイコン
  • 手動での残高調整

ローン

  • さまざまなローンタイプに対応：住宅ローン、自動車ローン、学生ローン、個人ローン、消費者ローン、クレジットカード
  • 詳細な返済スケジュール
  • 利息計算
  • 月々の支払いの追跡
  • 元利均等返済および元金均等返済の両方に対応
  • 自動支払い処理
  • 詳細なローン支払い記録
  • クレジットカードの定期的な利用の追跡

取引

  • 収入と支出を記録
  • 取引をカテゴリ分け
  • 定期的な取引に対応
  • 詳細な取引履歴
  • アカウント間の送金
  • カスタム取引日
  • 取引の検索とフィルタリング

データ管理

  • バックアップのために .regnskappdata ファイルへデータをエクスポート
  • 選択的データエクスポート（含めるデータを選択）
  • バックアップファイルからデータをインポート
  • インポート時にデータを置き換えるか結合するかを選択可能
  • 外部サーバーを使用しないローカルデータ保存

追加機能

  • 複数通貨に対応（160以上の通貨）
  • 多言語対応（アラビア語、中国語簡体字、中国語繁体字、デンマーク語、オランダ語、英語、フィンランド語、フランス語、ドイツ語、ヒンディー語、イタリア語、日本語、ノルウェー語、スペイン語、スウェーデン語、ウクライナ語）
  • アイコン付きのカスタムカテゴリ
  • 財務概要と統計
  • カスタマイズ可能な給料日設定
  • 生体認証（例：iOSのFace IDまたはTouch ID、Androidの指紋認証または顔認証）
  • カスタマイズ可能なオプション付きの「Make it Rain」アニメーション機能
  • 金額のセント表示のオン／オフ

よくある質問

新しいアカウントを追加するにはどうすればよいですか？

  1. Accounts タブに移動
  2. + ボタンをタップ
  3. アカウントの詳細を入力
  4. 保存

ローンを設定するにはどうすればよいですか？

  1. Loans タブに移動
  2. + ボタンをタップ
  3. ローンの種類を選択
  4. ローンの詳細を入力
  5. 保存

定期的な取引を設定するにはどうすればよいですか？

  1. 新しい取引を追加
  2. 「Recurring」トグルを有効にする
  3. 頻度を設定（毎週、毎月、毎年）
  4. 開始日と必要に応じて終了日を選択
  5. 取引を保存

バックアップのためにデータをエクスポートするにはどうすればよいですか？

  1. Settings に移動
  2. 「Export, Import & PDF」を選択
  3. 「Export」タブを選択
  4. 含めるデータを選択
  5. 「Export Data」をタップ
  6. .regnskappdata ファイルを保存する場所を選択

バックアップからデータをインポートするにはどうすればよいですか？

  1. Settings に移動
  2. 「Export, Import & PDF」を選択
  3. 「Import」タブを選択
  4. バックアップファイルを選択
  5. インポートを確認
  6. インポート前にすべての既存データが削除されます

言語を変更するにはどうすればよいですか？

  1. Settings に移動
  2. Language を選択
  3. 希望の言語を選択
  4. 必要に応じてアプリを再起動

生体認証ロックを有効にするにはどうすればよいですか？

iOS: ホーム画面でアプリアイコンを長押しし、「Require Face ID」または「Require Touch ID」を選択します。

Android: アプリを開き、Settings → Security または App lock に移動して、生体認証ロックを有効にします。

通貨を変更するにはどうすればよいですか？

  1. Settings に移動
  2. Currency をタップ
  3. リストから希望の通貨を選択

給料日を設定するにはどうすればよいですか？

  1. Settings に移動
  2. Payday 設定を見つける
  3. 通常の給料日を設定するように調整
  4. システムは週末を自動的に調整できます

データセキュリティ

  • すべてのデータはデバイスにローカル保存されます。iOSではSwiftDataを使用し、Androidではローカルデータベースを使用します
  • インターネット接続は不要です
  • アカウント作成は不要です
  • 生体認証に対応
  • 手動のデータエクスポートおよびインポートにより完全な管理が可能です
  • 財務データはデバイスの内蔵暗号化によって保護されています
