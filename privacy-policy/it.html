Informativa sulla privacy

Introduzione

BalanceTrackr (RegnskApp) rispetta la tua privacy. Questo documento spiega come gestiamo i tuoi dati.

Raccolta e archiviazione dei dati

  • Tutti i tuoi dati finanziari sono archiviati localmente sul dispositivo
  • Non raccogliamo, trasferiamo o archiviamo dati su server esterni
  • Nessuna informazione personale viene condivisa con terze parti
  • L'app non richiede connessione a Internet

Utilizzo dei dati

L'app utilizza i tuoi dati per:

  • Mostrare saldi e transazioni
  • Calcolare pagamenti e piani di ammortamento
  • Mostrare statistiche e grafici
  • Gestire transazioni ricorrenti

Archiviazione locale

  • I dati sono archiviati in modo sicuro con SwiftData
  • I dati rimangono sul dispositivo salvo tua eliminazione
  • Puoi eliminare tutti i dati disinstallando l'app

Autenticazione biometrica

  • Supporto per Face ID/Touch ID a livello di dispositivo
  • I dati biometrici sono gestiti da iOS; BalanceTrackr non vi accede
  • L'app non raccoglie n√© archivia dati biometrici

Esportazione e importazione dei dati

  • Esportazione in file .regnskappdata per backup
  • Controlli cosa viene esportato
  • I file esportati restano locali
  • Importazione dai backup per ripristino o trasferimento

Sicurezza

  • Protezione a livello dispositivo con Face ID/Touch ID
  • Nessuna autenticazione esterna
  • Nessun trasferimento di dati fuori dal dispositivo
  • Cifratura iOS a livello dispositivo
